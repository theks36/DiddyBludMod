# Diddy Blud Mod, for Minecraft NeoForge 1.21.1
Diddy Blud mod is created with MCreator, and it's purpose is to be a fun little personal project that I could see myself playing. It's nothing serious; neither are my MCreator skills.

See releases/downloads [here!](https://github.com/theks36/DiddyBludMod/releases)

## Features
- New ores, all of which are balanced to work with the vanilla progression system.
- New biome(s) with their own features, wood types, rock types, entities, ores.
- Other
